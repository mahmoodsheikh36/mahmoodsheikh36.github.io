a consequence of the definition of Euler’s Number is: \[ \lim_{x\to \pm\infty} \left(1+\frac{1}{f(x)}\right)^{f(x)} = e \iff f(x) \to \pm\infty \] thus, if we have an indeterminate form that can be rewritten in this form, it would be easy to calculate it

\[ \lim_{x\to \infty} \left(1+\frac{3}{x}\right)^x \] substitution of \(x=\infty\) give us \(1^\infty\) which is indeterminate, but since \(f(x)=\frac{x}{3}\to\infty\) we can use Euler’s Number to get its value we rewrite the limit as: \[ \lim_{x\to\infty} \left(1+\frac{1}{\frac{x}{3}}\right)^x \] now we need to move the 3 to the exponent, to do that we can multiply and divide the exponent by 3 \[ \lim_{x\to\infty} \left(1+\frac{1}{\frac{x}{3}}\right)^{x\cdot\frac{3}{3}} = \lim_{x\to\infty} \left(\underbrace{\left(1+\frac{1}{\frac{x}{3}}\right)^{\frac{x}{3}}}_{e}\right)^3 = e^3 \]

\[ \lim_{x \to \infty} \left( \frac{3x+1}{3x-5} \right)^{x-2} \] again, this is an indeterminate form \(1^\infty\). the first thing we can do to obtain the standard form is to add and subtract a quantity to the numerator of the fraction. in this case this quantity is 5 \[ \lim_{x\to\infty} \left(\frac{3x-5+5+1}{3x-5}\right)^{x-2} \] we can now split the fraction: \[ \lim_{x\to\infty} \left(\frac{3x-5}{3x-5} + \frac{6}{3x-5}\right)^{x-2} = \lim_{x\to\infty} \left(1+\frac{6}{3x-5}\right)^{x-2} \] at this point we need to “adjust” our fraction in order to have the standard form \[ \lim_{x\to\infty} \left(1+\frac{1}{\frac{3x+5}{6}}\right)^{x-2} \] finally, we multiply and divide the exponent by \(f(x)\) \[ \lim_{x\to\infty} \left(1+\frac{1}{\frac{3x+5}{6}}\right)^{(x-2) \cdot \frac{3x+5}{6} \cdot \frac{6}{3x+5}} \] which led us to: \[ \lim_{x\to\infty} \left(\left(1+\frac{1}{\frac{3x+5}{6}}\right)^{\frac{3x+5}{6}}\right)^{\frac{6x-12}{3x+5}} = e^\frac{6}{3} = e^2 \]

\[ \lim_{n\to\infty} \left(\frac{n^2-5n+6}{n^2-9n+20}\right)^{2n-1} \] in this case, we first need to divide the numerator by the denominator (i.e., polynomial long division). This leaves us with two new polynomials: \[ P(x)=1 \qquad R(x)=4n-14 \] we can rewrite the limit like this: \[ \lim_{n\to\infty} \left(1+\frac{4n-14}{n^2-9n+20}\right)^{2n-1} \] the rest is similar to the previous example \[ = \lim_{n\to\infty} \left(1+\frac{1}{\frac{n^2-9n+20}{4n-14}}\right)^{2n-1 \cdot \frac{n^2-9n+20}{4n-14} \cdot \frac{4n-14}{n^2-9n+20}} = \lim_{n\to\infty} \left( \underbrace{\left(1+\frac{1}{\frac{n^2-9n+20}{4n-14}}\right)^{\frac{n^2-9n+20}{4n-14}}}_{e}\right)^{\frac{(4n-14)(2n-1)}{n^2-9n+20}} = e^8 \]

5.9 L’Hôpital’s rule

see https://en.wikipedia.org/wiki/L%27H%C3%B4pital%27s_rule L’Hôpital’s rule states that for functions \(f\) and \(g\) which are differentiable on an open interval \(I\), except possibly at a point \(c\) contained in \(I\), if \(\lim_{x\to c} f(x) = \lim_{x\to c} g(x) = 0\) or \(\lim_{x\to c} f(x) = \lim_{x\to c} g(x) = \pm\infty\), and \(g’(x) \neq 0\) for all \(x\) in \(I\) with \(x \neq c\) and \(\lim_{x\to c} \frac{f’(x)}{g’(x)}\) exists then \[ \lim_{x\to c} \frac{f(x)}{g(x)} = \lim_{x\to c} \frac{f’(x)}{g’(x)} \] the differentiation of the numerator and denominator often simplifies the quotient or converts it to a limit that can be evaluated directly.

\[ \lim_{x\to \infty} x\sin\frac1x \] if we were to substitute \(x=\infty\) we would get: \[ \infty \cdot \sin\frac1\infty = \infty \cdot \sin0 = \infty \cdot 0 \] which is an indeterminate form we can rewrite the given expression as a fraction of 2 functions: \[ x\sin\frac1x =\frac{\sin\frac1x}{\frac1x} \] now we can substitute \(t=\frac{1}{x}\) to make our lives easier, so we get: \[ \lim_{x\to \infty} \frac{\sin\frac1x}{\frac1x} = \lim_{t\to 0} \frac{\sin{t}}{t} \] notice how the limit changed from \(\lim_{x\to \infty}\) to \(\lim_{t\to 0}\), this is because we substituted \(t=\frac{1}{x}\), and \(\lim_{x\to \infty} \frac{1}{x} = 0\) now since \(\lim_{t\to 0} \sin{t} = \lim_{t\to 0} t = 0\) we can make use of L’Hôpital’s rule \[ \lim_{t\to 0} \frac{\sin{t}}{t} = \lim_{t\to 0} \frac{\cos{t}}{1} = 1 \] and since this limit is equal to the original limit, we finally get: \[ \lim_{x\to \infty} x\sin\frac1x = 1 \]

7.16.1 direct comparison test

given \(f(x) \geq g(x) \geq 0\) and it follows that \(\int_1^\infty g(x) \text{ dx} \leq \int_1^\infty f(x) \text{ dx}\), if \(\int_1^\infty f(x) \text{ dx}\) converges then \(\int_1^\infty g(x) \text{ dx}\) converges too and if \(\int_1^\infty g(x) \text{ dx}\) diverges then \(\int_1^\infty f(x) \text{ dx}\) diverges too

if we wanted to know if the integral \(\int_1^\infty \frac{1}{x^3+7} \text{ dx}\) diverges or converges we look at the expression we arrived at in the last lemma we take the function \(\frac{1}{x^3}\) which corresponds to the first case in that lemma where \(a > 1\) which means this function converges and since we know \(\frac{1}{x^3+7} < \frac{1}{x^3}\) and \(\int_1^\infty \frac{1}{x^3} \text{ dx}\) converges then we know \(\int_1^\infty \frac{1}{x^3+7} \text{ dx}\) converges too

check if the following integral diverges/converges \[ \int_2^\infty \frac{1}{\ln{x}} \text{ dx} \] we consider the function \(\frac1x\), we know \(\ln{x} \leq x\) is always true, if we were to turn them into fractions we would get \(\frac{1}{\ln{x}} \geq \frac{1}{x}\) and if we look at the same lemma from before we know \(\int_2^\infty \frac{1}{x} \text{ dx}\) diverges because it corresponds to the case where \(a=1\) and since \(\int_2^\infty \frac{1}{x} \text{ dx}\) diverges and \(\int_2^\infty \frac{1}{x} \text{ dx} \leq \int_2^\infty \frac{1}{\ln{x}} \text{ dx}\) then we know \(\int_2^\infty \frac{1}{\ln{x}} \text{ dx}\) diverges

consider the integral: \[ \int_5^\infty \frac{1}{x\ln^2 x} \text{ dx} \] we use integration by substitution where \(t = \ln x\) \(\text{ dt} = \frac1x \text{ dx} \Longrightarrow \text{ dx} = x\text{ dt}\)

\begin{align*} \int \frac{1}{x\ln^2 x} \text{ dx} &= \int \frac{1}{xt^2} \cdot x \text{ dt}\\ &= \int \frac{1}{t^2} \text{ dt}\\ &= \int t^{-2} \text{ dt}\\ &= -t^{-1}\\ &= \frac{-1}{t}\\ &= \frac{-1}{\ln x}\\ \end{align*}

we substitute the interval:

\begin{align*} \int_5^\infty \frac{1}{x\ln^2 x} \text{ dx} &= \frac{-1}{\ln x} \Big|_5^\infty\\ &= \underbrace{\frac{-1}{\ln \infty}}_{\clap{diverges to 0}} - \frac{-1}{\ln 5}\\ &= \frac{1}{\ln 5} \end{align*}

we found that the given integral converges to \(\frac{1}{\ln 5}\) (meaning the area between the given function and the \(x\) axis is equal to \(\frac{1}{\ln 5}\)).

7.16.3 Dirichlet’s test

given an integrable function \(f(x)\) for all \(x\) such that: \[ \left|\int_a^b f(x) \text{ dx}\right| \leq M \] where \(M\) is some constant, and given some monotonically decreasing function \(g(x)\) such that \(\lim_{x\to\infty} g(x) = 0\) then the integral \(\int_a^\infty f(x) \cdot g(x) \text{ dx}\) converges

consider the integral: \[ \int_1^\infty \frac{\sin x}{x} \text{ dx} \] we use this test to determine whether this integral diverges or converges

\begin{gather*} \int_1^\infty \frac{\sin x}{x} \text{ dx} = \int_1^\infty \sin x \cdot \frac1x \text{ dx}\\ \left|\int_a^b \sin x \text{ dx}\right| = \left|-\cos x\big|_a^b\right| = |-\cos b + \cos a| \leq 2 \end{gather*}

we take \(g(x) = \frac1x\), we know \(\lim_{x\to\infty} g(x) = 0\) and \(g’(x)=-\frac1{x^2} \leq 0\) therefore \(g(x)\) is a monotonically decreasing function therefore we can use it in our Dirichlet test and therefore \(\int_a^\infty f(x) \cdot g(x) \text{ dx}\) converges therefore the original integral \(\int_1^\infty \frac{\sin x}{x} \text{ dx}\) converges

\begin{align*} \int_1^\infty \frac{\cos^2 x}{x} \text{ dx} &= \int_1^\infty \frac{1+\cos 2x}{2x} \text{ dx}\\ &= \frac12\int_1^\infty \frac{1}{x} \text{ dx} + \frac12\int_1^\infty \frac{\cos 2x}{x} \text{ dx}\\ \end{align*}

we take the second integral \(\int_1^\infty \frac{\cos 2x}{x} \text{ dx}\), this integral is of 2 functions:

\begin{gather*} \left|\int_a^b \cos 2x \text{ dx}\right| = \left|\frac{\sin 2x}{2}\Bigg|_a^b\right| = \left|\frac{\sin 2b}{2} - \frac{\sin 2a}{2}\right| \leq 1\\ g(x) = \frac1x,\ g’(x) = \frac{-1}{x^2} < 0,\ \lim_{x\to\infty} g(x) = 0 \end{gather*}

therefore the integral of the second function converges and the integral of the first function \(\frac1x\) diverges therefore the sum of those integrals converges

consider the series: \[ \sum_{n=1}^\infty \frac1{n(n+1)} = \sum_{n=1}^\infty \frac1n - \frac1{n+1} = \frac11 - \frac12 + \frac12 - \frac13 + \frac13 - \frac14 + \cdots + \frac1n - \frac1{n+1} = 1 - \frac{1}{n+1} \] this series converges to 1 as it ends with \(1 - \frac{1}{n+1}\)

8.2.1 direct comparison test

given the infinite series \(\sum^\infty b_n, \sum^\infty a_n\) such that \(|a_n| \leq |b_n|\) for sufficiently large \(n\) then:

  1. if the series \(b_n\) converges then the series \(a_n\) also converges
  2. if the series \(a_n\) diverges then the series \(b_n\) also diverges

\begin{align*} \sum^\infty \frac{1}{n^5+17} \leq& \sum^\infty \frac{1}{n^5}\\ \text{converges} \impliedby& \text{converges} \end{align*}

does this series converge or diverge? \[ \sum_{n=2}^\infty \frac{1}{\ln n} \] it diverges because \(\frac{1}{\ln x} > \frac{1}{n}\) and \(\sum \frac{1}{n}\) diverges

8.2.2 limit comparison test

if \(\{a_n\},\{b_n\} > 0\) (all elements of the two sequences are positive) and the limit \(\lim_{n\to\infty} \frac{a_n}{b_n}\) exists and is non-zero, then \(\sum_{n=1}^\infty a_n\) diverges if and only if \(\sum_{n=1}^\infty b_n\) diverges

\[ \sum \frac{3n^3+7n^2-2}{7n^5+4n^2-3} \] we divide by \(\frac{1}{n^2}\) \[ \lim_{n\to\infty} \frac{\frac{3n^3+7n^2-2}{7n^5+4n^2-3}}{\frac{1}{n^2}} = \lim_{n\to\infty} \frac{3n^5+7n^4-2n^2}{7n^5+4n^2-3} = \frac37 \] we know \(\sum \frac{1}{n^2}\) converges therefore \(\sum \frac{3n^3+7n^2-2}{7n^5+4n^2-3}\) converges too

8.2.3 Ratio test

suppose there exists \(r\) such that \[ \lim_{n\to\infty} \left|\frac{a_{n+1}}{a_n}\right| = r \] If \(r < 1\), then the series \(\sum a_n\) is convergent. If \(r > 1\), then the series is divergent. If \(r = 1\), the ratio test is inconclusive, and the series may converge or diverge, and we may use the root test which might succeed where the ratio test fails

consider \(\sum \frac1n\) which we already know is divergent, \(\lim_{n\to\infty} \frac{\frac{1}{n+1}}{\frac{1}{n}} = \lim_{n\to\infty} \frac{n}{n+1} = 1\) which means the test is inconclusive (fails)

consider \(\sum \frac{1}{3^n}\) \[ \frac{a_{n+1}}{a_n} = \frac{3^n}{3^{n+1}} = \frac{3^n}{3 \cdot 3^n} = \frac13 < 1 \] which means \(\sum \frac{1}{3^n}\) converges

consider the series: \[ \sum \frac{(n!)^2}{(2n)!} \] we use the ratio test

\begin{align*} \frac{a_{n+1}}{a_n} &= \frac{((n+1)!)^2}{(2(n+1))!} \cdot \frac{(2n)!}{(n!)^2}\\ &= \frac{(n+1)^2 \cdot (n!)^2}{(2n+2)!} \cdot \frac{(2n)!}{(n!)^2}\\ &= \frac{(n+1)^2 \cdot \cancel{(n!)^2}}{(2n+2)(2n+1)\cancel{((2n)!)}} \cdot \frac{\cancel{(2n)!}}{\cancel{(n!)^2}}\\ &= \frac{n^2+2n+1}{4n^2+6n+2}\\ &\implies \frac14 \end{align*}

which means this series is convergent

consider the series \[ \sum \frac{2^n \cdot n!}{n^n} \] we make use of the ratio test

\begin{align*} \frac{a_{n+1}}{a_n} &= \frac{2^{n+1} \cdot (n+1)!}{(n+1)^{n+1}} \cdot \frac{n^n}{2^n \cdot n!}\\ &= \frac{2 \cdot \cancel{2^n \cdot (n+1)n!} \cdot n^n}{\cancel{(n+1)}(n+1)^n \cdot \cancel{2^n \cdot n!}}\\ &= \frac{2n^n}{(n+1)^n}\\ &= 2\left(\frac{n}{n+1}\right)^n \end{align*}

we find the limit of \(\left(\frac{n}{n+1}\right)^n\) using euler

\begin{gather*} \lim_{n\to\infty} \left(\frac{n}{n+1}\right)^n = e^L\\ L = \lim_{n\to\infty} \left(\frac{n}{n+1} - 1\right)n = \lim_{n\to\infty} \left(\frac{-1}{n+1}\right)n = \lim_{n\to\infty} \frac{-n}{n+1} = -1 \end{gather*}

we continue where we left off: \[ \lim_{n\to\infty} 2\left(\frac{n}{n+1}\right)^n = 2e^{-1} = \frac{2}{e} < 1 \] which means this series is convergent

\[ \sum_{n=1}^\infty \sin\frac1n \] we know \(\sin x\) behaves like \(x\) so we divide by \(\frac1n\) and use the limit comparison test \[ \lim_{n\to\infty} \frac{\sin\frac1n}{\frac1n} = 1 \] which means the series is divergent because \(\sum \frac1n\) diverges note that \(\sum_{n=1}^\infty \sin^2\frac1n\) is convergent because we would divide by \(\frac1{n^2}\) whose series converges

8.2.4 Root test

let \[ r = \lim_{n\to\infty} \sqrt[n]{|a_n|} \] if \(r < 1\), the series converges, if \(r > 1\), then the series diverges, if \(r = 1\), the root test is inconclusive, and the series may converge or diverge.

this is a consequence of the following equality: \[ \lim \frac{a_{n+1}}{a_n} = \lim \sqrt[n]{a_n} \]

consider \[ \sum \frac{1}{3^n} \] we use the root test \[ \sqrt[n]{a_n} = \sqrt[n]{\frac{1}{3^n}} = \frac13 < 1 \] which means that the series converges

consider \[ \sum \frac{1}{\left(\frac{n+1}{n+3}\right)^{n(n+1)}} \] we apply the root test \[ \sqrt[n]{a_n} = \sqrt[n]{\frac{1}{\left(\frac{n+1}{n+3}\right)^{n(n+1)}}} = \frac{1}{\left(\frac{n+1}{n+3}\right)^{n+1}} \] we find the limit of the fraction:

\begin{gather*} \lim_{n\to\infty} \left(\frac{n+1}{n+3}\right)^{n+1} = e^L\\ L = \lim_{n\to\infty} \left(\frac{n+1}{n+3}-1\right)(n+1) = \lim_{n\to\infty} \left(\frac{-2}{n+3}\right)(n+1) = \lim_{n\to\infty} \frac{-2n-2}{n+3} = -2 \end{gather*}

we go back to the original limit:

\begin{gather*} \lim_{n\to\infty} \frac{1}{\left(\frac{n+1}{n+3}\right)^{n+1}} = \lim_{n\to\infty} \frac{1}{e^{-2}} = e^2 > 1 \end{gather*}

which means that the series diverges

\[ \sum \left(\frac{n}{n+1}\right)^{2n} \] we apply the root test

\begin{gather*} \lim_{n\to\infty} \left(\frac{n}{n+1}\right)^{2n} = \frac{1}{e^L}\\ a_n = \left(\frac{n}{n+1}\right)^{2n} \implies \frac{1}{e^L} \neq 0\\ L \implies \left(\frac{n}{n+1}-1\right)2n = \left(\frac{-1}{n+1}\right)2n = \frac{-2n}{n+1} \implies -2\\ \lim_{n\to\infty} \left(\frac{n}{n+1}\right)^{2n} = \frac{1}{e^{-2}} = e^2 > 1 \end{gather*}

which means that the series diverges

8.2.5 Integral test

let \(f(x) > 0\) then the integral \(\int_1^\infty f(x) \text{ dx}\) and the series \(\sum_{n=1}^\infty f(n)\) converge or diverge together

\[ \sum_{n=5}^\infty \frac{1}{n\ln n} \] we use the integral test and compare the series to the integral \(\int_5^\infty \frac{1}{x\ln x} \text{ dx}\)

\begin{gather*} t = \ln x\\ \text{dt} = \frac1x \text{ dx}\\ \text{dx} = x \text{ dt}\\ x=5 \implies t = \ln5\\ x=\infty \implies t=\ln\infty=\infty\\ \int_{\ln5}^\infty \frac{1}{xt}x \text{ dt} = \underbrace{\int_{\ln5}^\infty \frac{1}{t} \text{ dt}}_{\text{diverges}} \end{gather*}

which means that the series \(\sum_{n=5}^\infty \frac{1}{n\ln n}\) diverges too

using both ratio test and direct comparison test

\begin{gather*} \sum \frac{1!+2!+3!+\cdots+n!}{(2n)!} \leq \sum \frac{n!+n!+n!+\cdots+n!}{(2n)!} = \sum \frac{n\cdot n!}{(2n)!}\\ \frac{a_{n+1}}{a_n} = \frac{(n+1)(n+1)!}{(2(n+1))!} \cdot \frac{(2n)!}{n\cdot n!} = \frac{(n+1)\cancel{(n+1)n! \cdot (2n)!}}{\cancelto{2}{(2n+2)}(2n+1)\cancel{(2n)!} \cdot n \cdot \cancel{n!}} = \frac{n+1}{2(2n+1)n} = \frac{n+1}{4n^2+2n} \implies 0 < 1 \end{gather*}

which means the series converges

unfinished

\[ \sum_{n=7}^\infty = \frac{\ln^{-3} n}{n} \] we apply the integral test

\begin{gather*} t = \ln x\\ \text{dt} = \frac1x \text{ dx}\\ \text{dx} = x\text{ dt}\\ x=7 \implies t=\ln7\\ x=\infty \implies t=\ln\infty=\infty\\ \int_{\ln7}^\infty \frac{1}{x\ln^3 x} \text{ dx} = \int_{\ln7}^\infty \frac{1}{xt^3} x \text{ dt} = \underbrace{\int_{\ln7}^\infty \frac{1}{t^3} \text{ dt}}_{\text{converges}} \end{gather*}

which means the series converges too

\[ \sum \frac{(n+1)! + (n+2)! + \cdots + (2n)!}{(2n+3)!} \leq \sum \frac{n(2n)!}{(2n+3)(2n+2)(2n+1)(2n)!} \leq \sum \frac{n}{n \cdot n \cdot n} = \sum \frac{1}{n^2} \] and since \(\sum \frac{1}{n^2}\) converges we know the original series converges because its smaller

\[ \sum \frac{2^n}{\left(\frac{3n+2}{3n+1}\right)^{n(2n-3)}} \] we use the Root test: \[ \sqrt[n]{\frac{2^n}{\left(\frac{3n+2}{3n+1}\right)^{n(2n-3)}}} = \frac{2}{\left(\frac{3n+2}{3n+1}\right)^{2n-3}} \implies \frac{2}{e^{\frac23}} > 1 \] which means the series diverges

8.2.6 Raabe–Duhamel’s test

given \(\lim_{n\to\infty} \left(\frac{a_n}{a_{n+1}}-1\right)n = L\)

  1. if \(L < 1\) then \(\sum a_n\) diverges
  2. if \(L > 1\) then \(\sum a_n\) converges
  3. the test is inconclusive

we use this test when the Ratio test fails

\[ \sum \frac{n!}{(5+1)(5+2)(5+3) \cdots (5+n)} \] we try Ratio test:

\begin{align*} \frac{a_{n+1}}{a_n} &= \frac{(n+1)!}{(5+1)(5+2) \cdots (5+n+1)} \cdot \frac{(5+1)(5+2) \cdots (5+n)}{n!}\\ &= \frac{(n+1)n!}{(5+1)(5+2) \cdots (5+n+1)} \cdot \frac{(5+1)(5+2) \cdots (5+n)}{n!}\\ &= \frac{n+1}{n+6}\\ &\implies 1 \end{align*}

Ratio test failed because we got 1 so we apply Raabe’s test:

\begin{align*} \lim_{n\to\infty} \left(\frac{a_n}{a_{n+1}}-1\right)n &= \lim_{n\to\infty} \left(\frac{n+6}{n+1}-1\right)n\\ &= \lim_{n\to\infty} \left(\frac{5}{n+1}\right)n\\ &= \lim_{n\to\infty} \frac{5n}{n+1}\\ &= 5 > 1 \end{align*}

we got \(5 > 1\) which means the series converges

8.2.7 Alternating series test

suppose the following statements are true:

  1. \(a_{n}\) are all positive,
  2. \(\lim_{n\to\infty} a_n = 0\)
  3. for every \(n\), \(a_{n+1} \leq a_n\)

then \(\sum_{n=k}^{\infty} (-1)^n a_n\) and \(\sum_{n=k}^{\infty} (-1)^{n+1} a_n\) are convergent series

\[ \sum_{n=1}^{\infty} \frac{(-1)^n}{n} = \sum_{n=1}^{\infty} (-1)^n\frac{1}{n} \] we check the conditions of the Alternating series test

  1. \(\frac{1}{n} \geq 0\)
  2. \(\lim_{n\to \infty} \frac{1}{n} = 0\)
  3. \(\frac{1}{n+1} < \frac{1}{n}\)

and since \(\frac{1}{n}\) meets all the conditions, we can apply the test to \(\sum_{n=1}^{\infty} (-1)^n\frac{1}{n}\) and find that it converges
