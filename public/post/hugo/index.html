1 redirect 404 to homepage

to do this we create a file 404.html in our theme’s layout directory with the following content: 

{{ define "content"}}
<meta http-equiv="refresh" content="0; url=https://blog.mahmoodsheikh.com" />
<main id="main">
  <div>
    <!-- for older browsers that cant handle redirects -->
    <h1 id="title"><a href="{{ "/" | relURL }}">Go Home</a></h1>
  </div>
</main>
{{ end }}

hugo by default wont use this file, i use nginx so to make the webserver return this file on 404 errors we add the line error_page 404 /404.html; to /etc/nginx/sites-available/blog
and then use sudo systemctl restart nginx to restart nginx with the new configuration
