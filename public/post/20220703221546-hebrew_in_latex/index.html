hebrew in latex

January 1, 0001

you need to install the package texlive-most, if you havent already

this is the only approach that worked for me on arch linux install the texlive-langextra package with: 

sudo pacman -S texlive-langextra

use the following basic template: 

  \documentclass[12pt]{article}
  \usepackage[english,bidi=basic]{babel} %bidi=default with xelatex
  \babelprovide[import,main]{hebrew}

  \babelfont{rm}[Language=Default]{Latin Modern Roman}
  \babelfont[hebrew]{rm}{DejaVu Sans}

  \begin{document}

  \begin{center}
    \Huge
    \textbf{תקציר}
  \end{center}

ניסוי ותהייה, או ניסוי וטעיה?

  \selectlanguage{english} some text 

  \end{document}

compile the file using lualatex with lualatex file.tex, since neither standard latex (pdflatex) nor xelatex worked for me
