heap sort

heap sort consists of transforming the given array into a heap array A, initiating heap-size to size(A) then applying the following steps recursively:

  1. replace the root of the heap A[0] with A[heap-size - 1]
  2. bubble down the newly replaced root (A[0])
  3. decrement heap-size, return to step 1 if heap-size > 1
