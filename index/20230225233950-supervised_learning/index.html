supervised learning

cite:&aima2010 the task of supervised learning is this:

given a training set of example input-output pairs , where each was generated by an unknown function , discover a function that approximates the true function .

here and can be any value; they need not be numbers. the function is a hypothesis from a hypothesis space, denoted by , learning is a search through the space of possible hypotheses for one that will perform well, even on new examples beyond the training set. to measure the accuracy of a hypothesis we give it a test set of examples that are distinct from the training set. we say a hypothesis generalizes well if it correctly predicts the value of for novel examples. sometimes the function is stochastic–it is not strictly a function of , and what we have to learn is a conditional probability distribution, When the output is one of a finite set of values (such as sunny, cloudy or rainy), the learning problem is called classification, and is called boolean classification if there are only two values. When is a number (such as tomorrow’s temperature), the learning problem is called regression.
