binary search

a search algorithm goal: given a sorted array and a key, find index of the key in the array method: compare key against middle entry

  • too small, go left
  • too big, go right
  • equal, found

time complexity is 

public static void main(String[] args) {
  int[] a = {1,10,11,21,30};
  System.out.println(binarySearch(a, 10));
}

static int binarySearch(int[] a, int key) {
  int lo = 0, hi = a.length - 1;
  while (lo <= hi) {
    int mid = lo + (hi - lo) / 2;
    if (key < a[mid]) hi = mid - 1;
    else if (key > a[mid]) lo = mid + 1;
    else return mid;
  }
  return -1;
}
