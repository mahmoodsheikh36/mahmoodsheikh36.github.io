make CFLAGS='-m32 -g'

we break at the line 362 in the file variables.c which is the following:

parse_and_execute (temp_string, name, SEVAL_NONINT|SEVAL_NOHIST);

we see that the value of temp_string equals "sayh () { echo \"it works\"; }", which is then passed to parse_and_execute to be executed, which results in the function sayh being defined, as would happen in any bash version, e.g. (at the time of writing) in the most recent version on arch linux:

echo $BASH_VERSION
sayh () { echo \"it works\"; }
sayh
