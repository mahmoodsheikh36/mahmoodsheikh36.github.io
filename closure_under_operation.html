closure under operation

a set is closed under an operation if applying that operation to any member of the set always yields a member of the set.

if a set is closed under a binary operation, it doesnt mean that the operands for a given result always exist, consider the set of integers which is closed under addition and contains 1, and the equation 1/3+2/3=1, neither operand is in the set of integers.
