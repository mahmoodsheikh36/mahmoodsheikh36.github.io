<div class="dummy" id="def-alphabet">

an _alphabet_ is a [finite](20231015T221141--finite-set__.org), non-empty [set](20240205T193039--set__math.org) of [symbol](20240205T193014--symbol__.org)s. conventionally, we use the symbol <img src="/ltximg/4ddbce7ed65.svg" alt="\(\Sigma\)" style="height: 0.7667em; vertical-align: -0.0502em; display: inline-block" class="org-latex org-latex-inline" /> for an alphabet. common alphabets include: <br/>

1.  <img src="/ltximg/391fc027dca.svg" alt="\(\Sigma=\{0, 1\}\)" style="height: 1.0294em; vertical-align: -0.2697em; display: inline-block" class="org-latex org-latex-inline" />, the _binary alphabet_. <br/>
2.  <img src="/ltximg/20aac59c364.svg" alt="\(\Sigma=\{a,b,\dots,z\}\)" style="height: 1.0294em; vertical-align: -0.2697em; display: inline-block" class="org-latex org-latex-inline" />, the set of all lower-case letters. <br/>
3.  the set of all ASCII characters, or the set of all printable ASCII characters. <br/>

(John E. Hopcroft, Rajeev Motwani, Jeffrey D. Ullman, 2006) <br/>

</div>

<div class="corollary">

[set operations](20231201T190410--set-operations__.org) apply to alphabets. <br/>
