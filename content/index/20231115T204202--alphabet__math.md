<div class="dummy" id="def-alphabet">

an _alphabet_ is a [finite](20231015T221141--finite-set__math.org), non-empty [set](20240205T193039--set__math.org) of [symbol](20240205T193014--symbol__.org)s. conventionally, we use the symbol <img src="/ltximg/400e9daadfe.svg" alt="\(\Sigma\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> for an alphabet. common alphabets include: <br/>

1.  <img src="/ltximg/55f4336a9b7.svg" alt="\(\Sigma=\{0, 1\}\)" style="height: 1.0784em; vertical-align: -0.2942em; display: inline-block" class="org-latex org-latex-inline" />, the _binary alphabet_. <br/>
2.  <img src="/ltximg/dbbc744a545.svg" alt="\(\Sigma=\{a,b,\dots,z\}\)" style="height: 1.0784em; vertical-align: -0.2942em; display: inline-block" class="org-latex org-latex-inline" />, the set of all lower-case letters. <br/>
3.  the set of all ASCII characters, or the set of all printable ASCII characters. <br/>

(John E. Hopcroft, Rajeev Motwani, Jeffrey D. Ullman, 2006) <br/>

</div>

<div class="corollary">

[set operations](20231201T190410--set-operations__.org) apply to alphabets. <br/>

</div>

if <img src="/ltximg/400e9daadfe.svg" alt="\(\Sigma\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> is an alphabet, we can express the set of all strings of a certain a certain length from that alphabet by using an exponential notation. we define <img src="/ltximg/415dea42ccc.svg" alt="\(\Sigma^k\)" style="height: 0.9304em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> to be the set of strings of length <img src="/ltximg/ce369eb3609.svg" alt="\(l\)" style="height: 0.7789em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" />, each of whose symbols is in <img src="/ltximg/400e9daadfe.svg" alt="\(\Sigma\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" />. <br/>
the set of all strings over an alphabet <img src="/ltximg/400e9daadfe.svg" alt="\(\Sigma\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> is conventionally denoted <img src="/ltximg/28aa3b9314e.svg" alt="\(\Sigma^*\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" />. for instance, <img src="/ltximg/48bdf188125.svg" alt="\(\{O, 1\}^*=\{\varepsilon,0,1,00,01,10,11,000,\dots\}\)" style="height: 1.0784em; vertical-align: -0.2942em; display: inline-block" class="org-latex org-latex-inline" />. put another way, <br/>
<img src="/ltximg/54614afa30a.svg" alt="\[ \Sigma^* = \Sigma^{0} \cup \Sigma^{1} \cup \Sigma^{2} \cup \dots \]" style="height: 1.2744em; display: block" class="org-latex org-latex-block" /> <br/>
sometimes we wish to exclude the empty string from te set of strings. the set of nonempty strings from alphabet <img src="/ltximg/400e9daadfe.svg" alt="\(\Sigma\)" style="height: 0.7735em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> is denoted <img src="/ltximg/bce942796d0.svg" alt="\(\Sigma^+\)" style="height: 0.8779em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" />. thus, two appropriate equivalences are: <br/>
