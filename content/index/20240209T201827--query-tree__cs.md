<div class="dummy">

a query tree is a [tree data structure](20240201T173023--tree-data-structure__math.org) that corresponds to an extended [relational algebra](20231212T163612--relational-algebra__math.org) expression. it represents the input relations of the query as [leaf](20240213T202734--leaf__cs.org) nodes of the tree, and it represents the relational algebra operations as internal nodes. an execution of the query tree consists of executing an internal node operation whenever its operands are available and then replacing that internal node by the relation that results from executing the operation. the order of execution of operations starts at the leaf nodes, which represents the input database relations for the query, and ends at the root node, which represents the final operation of the query. the execution terminates when the root node operation is executed and produces the result relation for the query. <br/>
(Ramez Elmasri, Shamkant B. Navathe, 2015 chapter 19.1.1 notation for query trees and query graphs) <br/>

</div>

```sql
SELECT Pnumber, Dnum, Lname, Address, Bdate
FROM   PROJECT, DEPARTMENT, EMPLOYEE
WHERE  Dnum = Dnumber AND Mgr_ssn = Ssn AND
       Plocation = 'Stafford'
```
