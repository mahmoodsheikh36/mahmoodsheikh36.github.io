it is convenient to denote a point in the graph of a function <img src="/ltximg/85e02a24e05.svg" alt="\(\brm{f}: \mathbb{R}^k \to \mathbb{R}^{n-k}\)" style="height: 0.9304em; vertical-align: -0.0492em; display: inline-block" class="org-latex org-latex-inline" /> as <img src="/ltximg/9944529a72e.svg" alt="\(\pmqty{\brm{x}\\ \brm{f}(\brm{x})}\)" style="height: 2.4504em; vertical-align: -0.5392em; display: inline-block" class="org-latex org-latex-inline" />. with <img src="/ltximg/91ab02ab2ff.svg" alt="\(\brm{x} \in \mathbb{R}^k\)" style="height: 0.9687em; vertical-align: -0.0875em; display: inline-block" class="org-latex org-latex-inline" /> and <img src="/ltximg/12ac318a508.svg" alt="\(\brm{f}(\brm{x}) \in \mathbb{R}^{n-k}\)" style="height: 1.1754em; vertical-align: -0.2942em; display: inline-block" class="org-latex org-latex-inline" />. but this presupposes that the "active" variables are the first variables, which is a problem, since we usually cannot use the same active variables at all points of the mainfold. <br/>
(john h. hubbard, barbara burke hubbard, 2015) <br/>
