<div class="definition">

with a predictive hypothesis <img src="/ltximg/353e59502b8.svg" alt="\(h\)" style="height: 0.8086em; vertical-align: -0.0568em; display: inline-block" class="org-latex org-latex-inline" /> and a target hypothesis <img src="/ltximg/df01f60c7cb.svg" alt="\(y\)" style="height: 0.8138em; vertical-align: -0.3157em; display: inline-block" class="org-latex org-latex-inline" />, the least squares <b>loss function</b> is defined by <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/a1157409a37.svg" alt="\begin{equation}
  L_2(\brm{w}) \defeq \sum_{i} \norm{h(\brm{x}_i)-\brm{y}_i}_2^2
\end{equation}
" style="height: 2.7248em; vertical-align: -1.3792em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

where <img src="/ltximg/246728fb788.svg" alt="\((\brm{x}_i, \brm{y}_i) \in \{(\brm{x},\brm{y})\}\)" style="height: 1.1165em; vertical-align: -0.3411em; display: inline-block" class="org-latex org-latex-inline" /> such that the latter is the set of observations from the target hypothesis <img src="/ltximg/df01f60c7cb.svg" alt="\(y\)" style="height: 0.8138em; vertical-align: -0.3157em; display: inline-block" class="org-latex org-latex-inline" />. <br/>

</div>

<div data-title="gradient of a least-squares loss in a linear model" class="my_example">

consider the <b>linear model</b> <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/6f85837ddc2.svg" alt="\begin{equation}
  \brm{y} = \brm{\Phi}\brm{\theta}^\intercal.
\end{equation}
" style="height: 1.5233em; vertical-align: -0.2981em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

where <img src="/ltximg/2940470b808.svg" alt="\(\brm{\theta} \in \mathbb{R}^{D}\)" style="height: 0.9778em; vertical-align: -0.0727em; display: inline-block" class="org-latex org-latex-inline" /> is a parameter space, <img src="/ltximg/7a1eff71149.svg" alt="\(\brm{\Phi} \in \mathbb{R}^{N \times D}\)" style="height: 0.9631em; vertical-align: -0.0580em; display: inline-block" class="org-latex org-latex-inline" /> are input features and <img src="/ltximg/3bb3f5cd663.svg" alt="\(\brm{y} \in \mathbb{R}^N\)" style="height: 1.2032em; vertical-align: -0.2981em; display: inline-block" class="org-latex org-latex-inline" /> are the corresponding observations. we define the functions <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/35df49c8b14.svg" alt="\begin{align}
  L(\brm{e}) &amp;amp;\defeq \norm{\brm{e}}^2,\\
  \brm{e}(\brm{\theta}) &amp;amp;\defeq \brm{y}-\brm{\Phi}\brm{\theta}.
\end{align}
" style="height: 3.6907em; vertical-align: -0.4020em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

we seek <img src="/ltximg/40304b19769.svg" alt="\(\frac{\partial L}{\partial \brm{\theta}}\)" style="height: 1.4600em; vertical-align: -0.4601em; display: inline-block" class="org-latex org-latex-inline" />, and we will use the <b>chain rule</b> for this purpose, <img src="/ltximg/76e8fedbbab.svg" alt="\(L\)" style="height: 0.7640em; vertical-align: -0.0568em; display: inline-block" class="org-latex org-latex-inline" /> is the least-squares loss function. <br/>
before we start our calculation, we determine the dimensionality of the derivative as <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/b97bb34c80b.svg" alt="\begin{equation}
  \frac{\partial L}{\partial \brm{\theta}} \in \mathbb{R}^{D}.
\end{equation}
" style="height: 2.2985em; vertical-align: -0.7450em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

the chain rule allows us to compute the <b>derivative</b> as <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/c50098fcce0.svg" alt="\begin{equation}
  \frac{\partial L}{\partial \brm{\theta}} = \frac{\partial L}{\partial \brm{e}}\frac{\partial \brm{e}}{\partial \brm{\theta}},
\end{equation}
" style="height: 2.2985em; vertical-align: -0.7450em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

where the <img src="/ltximg/0400ee8e1d7.svg" alt="\(d\)" style="height: 0.8086em; vertical-align: -0.0568em; display: inline-block" class="org-latex org-latex-inline" />th element is given by <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/1a6e3db2b39.svg" alt="\begin{equation}
  \frac{\partial L}{\partial \brm{\theta}}[1,d] = \sum_{n=1}^{N} \frac{\partial L}{\partial \brm{e}}[1,n]\frac{\partial \brm{e}}{\partial \brm{\theta}}[n,d].
\end{equation}
" style="height: 3.3323em; vertical-align: -1.3556em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

we know that <img src="/ltximg/829b3e71d89.svg" alt="\(\norm{\brm{e}}^2=\brm{e}^\intercal \brm{e}\)" style="height: 1.2104em; vertical-align: -0.2574em; display: inline-block" class="org-latex org-latex-inline" /> and determine <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/31b9574e92d.svg" alt="\begin{equation}
  \frac{\partial L}{\partial \brm{e}} = 2\brm{e}^\intercal \in \mathbb{R}^{1 \times N}
\end{equation}
" style="height: 2.2926em; vertical-align: -0.7391em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

furthermore, we obtain <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/745f4706c7c.svg" alt="\begin{equation}
  \frac{\partial \brm{e}}{\partial \brm{\theta}} = -\brm{\Phi} \in \mathbb{R}^{N \times D}.
\end{equation}
" style="height: 2.2985em; vertical-align: -0.7450em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

such that our desired derivative is <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/4708ecca150.svg" alt="\begin{equation}
  \frac{\partial L}{\partial \brm{\theta}} = -2\brm{e}^\intercal\brm{\Phi} = -2\underbrace{(\brm{y}^\intercal-\brm{\theta}^\intercal\brm{\Phi}^\intercal)}_{1 \times N}\underbrace{\brm{\Phi}}_{N \times D} \in \mathbb{R}^{1 \times D}.
\end{equation}
" style="height: 3.6948em; vertical-align: -2.1413em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

we would have obtained the same result without using the chain rule by immediately looking at the function <br/>


<div class="equation-container">
<span class="equation">
<img src="/ltximg/36a4ffb9456.svg" alt="\begin{equation}
  L_2(\brm{\theta}) \defeq \norm{\brm{y}-\brm{\Phi}\brm{\theta}}^2 = {(\brm{y}-\brm{\Phi}\brm{\theta})}^\intercal(\brm{y}-\brm{\Phi}\brm{\theta}).
\end{equation}
" style="height: 1.6437em; vertical-align: -0.2981em; display: inline-block" class="org-latex org-latex-inline" />
</span>
</div>

this approach is practical for functions like <img src="/ltximg/8567465baee.svg" alt="\(L_2\)" style="height: 0.9637em; vertical-align: -0.2564em; display: inline-block" class="org-latex org-latex-inline" /> but becomes impratical for deep function compositions. <br/>
(A. Aldo Faisal, Marc Peter Deisenroth, and Cheng Soon Ong, 2020 example 5.11) <br/>

</div>
