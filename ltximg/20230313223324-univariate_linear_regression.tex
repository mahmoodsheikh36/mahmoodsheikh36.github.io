\cite{aima2010}\\
a \textbf{univariate linear function} (a straight line) with input \(x\) and output \(y\) has the form \(y=w_1x+w_0\), where \(w_0\) and \(w_1\) are real-valued coefficients to be learned. we use the letter \(w\) because we think of the coefficients as \textbf{weights}; the value of \(y\) is changed by changing the relative weight of one term or another. we’ll define \(w\) to be the \href{linear_algebra2.org}{vector} \([w_0,w_1]\), and define\\
\[
h_{\brm{w}}(x)=w_1x+w_0
\]\\
the task of finding the \(h_{\boldsymbol{\mathrm{w}}}\) that best fits these data is called \textbf{linear regression}. to fit a line to the data, all we have to do is find the values of the weights \([w_0,w_1]\) that minimize the empirical loss. it is traditional to use the \href{20230420162447-squared_loss_function.org}{squared loss function} summed over all the training examples:\\
\[
Loss(h_{\boldsymbol{\mathrm{w}}}) = \sum_{j=1}^{N} \left(y_j-h_{\boldsymbol{\mathrm{w}}}(x_j)\right)^2 = \sum_{j=1}^{N} \left(y_j-(w_1x_j+w_0)\right)^2
\]\\
we would like to find \(\brm{w}=\argmin_w Loss(h_{\brm{w}})\). the sum \(\sum_{j=1}^{N} (y_j-(w_1x_j+w_0))^2\) is minimized when its partial derivatives with respect to \(w_0\) and \(w_1\) are zero:\\
\[
\frac{\partial}{\partial w_0} \sum_{j=1}^{N} (y_j-(w_1x_j+w_0))^2 = 0 \textbf{ and } \frac{\partial}{\partial w_1} \sum_{j=1}^{N} (y_j-(w_1x_j+w_0))^2=0
\]\\
these equations have a unique solution:\\
\[
w_1=\frac{N(\sum x_jy_j)-(\sum x_j)(\sum y_j)}{N(\sum x^2_j)-(\sum x_j)^2} \qquad w_0=(\sum y_j-w_1(\sum x_j))/N
\]
