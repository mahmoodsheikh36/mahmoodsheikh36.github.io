given a \href{data_structures.org}{binary tree}, write an \href{20220706211958-algorithm.org}{algorithm} that finds the maximum sum of a path from the root to the leaves such that no two consecutive nodes (parent and child) can be added to the sum\\[0pt]
\begin{answer}
we add two variables two each node, \texttt{with} and \texttt{without}, such that \texttt{with} is the route from the node with the maximal sum, and \texttt{without} is the route with the maximal sum starting from one of its children (node itself isnt part of the sum)\\[0pt]

\begin{question}
given an array of \href{calculus2.org}{normal} numbers of size \(n\) that represent prices of toys. a kidergarten teacher is interested in buying as many toys as possible with a budget of \(k\)\\[0pt]
\begin{subquestion}
write an algorithm that solves the problem in \(\Theta(n)\) time\\[0pt]
\includesvg{/Users/mahmooz/brain/out/UqiZP5F}\\[0pt]
\end{subquestion}
\begin{subquestion}
explain the algorithm and show its correctness\\[0pt]
\begin{answer}
the algorithm (almost) halves the array on each call until it finds the subarray of numbers to best add up to \(k\)\\[0pt]
\end{answer}
\end{subquestion}
\begin{subquestion}
analyze the time complexity of the algorithm\\[0pt]
the \href{20221105001640-recursive_function.org}{recurrence formula} for this algorithm is \(T(n)=Cn+T\left(\frac{n}{2}\right)\) which simplifies to \(T(n) \in \Theta(n)\)\\[0pt]
\end{subquestion}
\end{question}
