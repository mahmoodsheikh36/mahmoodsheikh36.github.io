the function \texttt{org-open-at-point} opens the link in another buffer by default, using \texttt{find-file-other-window}\\[0pt]
to change that use the following snippet\\[0pt]
\begin{lstlisting}[language=Lisp,label= ,caption= ,captionpos=b,numbers=none]
(setf (cdr (assoc 'file org-link-frame-setup)) 'find-file)
\end{lstlisting}
