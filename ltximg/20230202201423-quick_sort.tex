\textbf{QuickSort} is a \href{20220706211939-divide_and_conquer_algorithm.org}{divide-and-conquer} \href{20230202201515-sorting_algorithm.org}{sorting algorithm}, it picks an element as \textbf{pivot} and \href{discrete_maths2.org}{partition}s the given array around the pivot, there are different versions of quicksort that pick pivots in different ways:\\
\begin{enumerate}
\item first element as pivot (implemented here)\\
\item last element as pivot\\
\item a random element as pivot\\
\item median as pivot\\
\end{enumerate}
the first subarray we use as a "partition" is the array itself, where given a pivot, put the pivot at its correct position in the sorted array by putting all the smaller elements before it, and all the greater elements after it. then divide the partition in 2 partitions, where the first partition is where all the smaller elements were put, and the second partition is where all the greater elements were put, and apply the same process for these partitions recursively until we reach partitions that contain 1 element only\\
this algorithm runs in \(O(n\log n)\) time\\

template <typename T>
void swap(T arr[], int i, int j) {
  T tmp = arr[i];
  arr[i] = arr[j];
  arr[j] = tmp;
}

template <typename T>
int partition(T arr[], int low, int high) {
  int pivot = low++;
  while (low <= high) {
    if (arr[low] <= arr[pivot]) low++;
    else if (arr[high] >= arr[pivot]) high--;
    else swap(arr, low, high);
  }
  swap(arr, high, pivot);
  return high;
}

template <typename T>
void quick_sort(T arr[], int low, int high) {
  if (low <= high) {
    int pivot = partition(arr, low, high);
    quick_sort(arr, low, pivot-1);
    quick_sort(arr, pivot+1, high);
  }
}
\end{lstlisting}
