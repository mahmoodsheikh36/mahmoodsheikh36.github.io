to delete a file without leaving a trace use\\
\lstset{language=bash,label= ,caption= ,captionpos=b,numbers=none}
\begin{lstlisting}
shred -uz <file>
\end{lstlisting}
to keep hard drive clean run\\
\lstset{language=bash,label= ,caption= ,captionpos=b,numbers=none}
\begin{lstlisting}
cat /dev/zero > zero.file
sync
rm zero.file
\end{lstlisting}
then remove \texttt{zero.file}, this will fill the free space of the hard drive with null data
