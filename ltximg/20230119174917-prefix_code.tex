\begin{definition}
\textbf{prefix code} is \href{20230119160930-binary_coding.org}{binary code} in which no character is a prefix of another\\[0pt]
\end{definition}
\begin{note}
in \href{20230119170007-fixed_length_code_word.org}{fixed-length code word}, the codes are of the same length, so none of the characters can be a prefix of another, so any fixed-length code is by definition prefix code\\[0pt]
\end{note}
