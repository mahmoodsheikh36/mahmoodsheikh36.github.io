\textbf{heap sort} consists of \href{data_structures.org}{transforming the given array into a heap array} \texttt{A}, initiating \texttt{heap-size} to \texttt{size(A)} then applying the following steps recursively:\\
\begin{enumerate}
\item replace the root of the heap \texttt{A[0]} with \texttt{A[heap-size - 1]}\\
\item \href{data_structures.org}{bubble down} the newly replaced root (\texttt{A[0]})\\
\item decrement \texttt{heap-size}, return to step 1 if \texttt{heap-size > 1}\\
\end{enumerate}
\begin{lemma}
the \href{20221130014441-time_complexity.org}{time complexity} of this \href{20220706211958-algorithm.org}{algorithm} is \(\Theta(n\log n)\)\\
\end{lemma}
\begin{note}
on the third step when decrementing \texttt{heap-size} we arent deleting the last element, we are simply decreasing the size of the part of the list that we're operating on because the part at the end contains elements that are sorted\\
\end{note}
