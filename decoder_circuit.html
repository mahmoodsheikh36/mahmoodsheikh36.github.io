decoder circuit

a decoder is a combinational circuit with inputs and at most outputs. its characteristic property is that for every combination of input values, only one output value will be equal to 1 at any given time.
a decoder with exactly inputs and outputs can also be used to implement any switching function. each output of such a decoder realizes one distinct minterm. thus, by connecting the appropriate outputs to an OR gate, the required function can be realized.
it may have an enable input which turns the circuit on when it assumes the value 0.

one type of decoders converts the given binary input to a decimal output (so to speak).

the input is gave us the output (the index of the bit of value 1 is the decimal value, starting from the right)
