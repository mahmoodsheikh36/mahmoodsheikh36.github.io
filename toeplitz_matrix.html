toeplitz matrix

a Toeplitz matrix is a matrix in which values along each diagonal are all equal


an toeplitz matrix' elements are determined by a sequence of length :

so, given a sequence we can create a toeplitz matrix by the following steps:

  1. put the sequence in the first column of the matrix
  2. shift it and put it in the next column. when shifting, the last element disappears and a new element of the sequence appears. if there is no such new element, put a zero in that location.
