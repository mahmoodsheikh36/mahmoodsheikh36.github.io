first-order language

a first-order language contains the following symbols.

  1. the propositional connectives and , and the universal quantifier symbol .
  2. punctuation marks: the left parenthesis "(", the right parenthesis ")", and the comma ",".
  3. denumerably many individual variables
  4. a finite or denumerable, possibly empty, set of function letters.
  5. a finite or denumerable, possibly empty, set of individual constants.
  6. a nonempty set of predicate letters.
