maxima in lisp

i ended up giving up on using maxima as a visualization/symbolic math computation library for lisp, it wasnt made with this usecase in mind i guess, and so its an utter nightmare to lisp with it maxima is written in lisp and so it can be used as a modified/extended lisp runtime 1 how to to start the lisp runtime use the command: rmaxima -r 'to_lisp();' can also run this in emacs’ slime/sly with (sly "rmaxima -r to_lisp();") alternatively, we can clone maxima’s source code to the subdirectory local-projects of the quicklisp installation directory (usually ~/quicklisp unless you’ve modified it), follow the instructions at https://sourceforge.
