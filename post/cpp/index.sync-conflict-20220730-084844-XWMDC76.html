1.1 Segmentation fault

the infamous Segmentation fault error is caused by accessing or attempting to operate on an illegal memory address

1.2 stack variables

be careful when passing a pointer to a variable stored on the stack, e.g. the following program would fail because before the program exits the tree tries to delete its nodes but we would get a double free or corruption (out) error because its not safe to use delete on stack variables
