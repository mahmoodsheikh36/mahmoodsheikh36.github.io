find a boolean expression for the function

\begin{gather*} f(x,y,z) = x’y’z’+x’y’z+x’yz+xyz’\\ f(x,y,z) = (x+y’+z)(x’+y+z)(x’+y+z’)(x’+y’+z’) \end{gather*}

check whether \(f(x,y,z)\) is functionally complete

\begin{gather*} f(x,y,z) = x’y’z’+x’y’z+x’yz+xyz’\\ f(x,x,x) = x’x’x’ + x’x’x + x’xx + xxx’ = x’ + 0 + 0 + 0 = x’\\ f(x’,0,z) = x’‘0z’ + x’‘0’z’ + x’‘0z + x'0z = 0 + xz + 0 + 0 = xz \end{gather*}

negation is derived from the second equation and the operator \(AND\) is derived from the second
\(f(x,y,z)\) is functionally complete

check whether the following systems are functionally complete

\[ f_1(x,y) = x \oplus y \qquad f_2(x,y) = x + y’ \]

\(f_1(x,y)=x’y+xy’\)
\(f_1(x,1)=x’ \longrightarrow\) derived negation
\(f_2(x, f_2(0, y)) = x+y \longrightarrow\) derived OR operator
this system partially complete since i used 0/1 (using the set of operands \(\lnot\), \(\lor\))

\[ f_1(a,b,c) = a(b+c) \qquad f_2(a) = a’ \]

\(f_2(a) = a’ \longrightarrow\) derived negation
\(f_1(a,b,a’)=a(b+a’)=ab+aa’=ab+0=ab \longrightarrow\) derived AND operator
since we derived both negation and the AND operator (\(\lnot, \land\)) using the system then its functionally complete

\(f_1(a,b)=a’b+ab’\)
\(f_2(a,b)=a’b\)
not a functionally complete system
