algorithm correctness

an algorithm is totally correct if it receives valid input, terminates, and always returns the correct output

the only way to prove the correctness of an algorithm over all possible inputs is by reasoning formally or mathematically about it.
generally, to prove an algorithm is correct we prove two things

  1. the algorithm terminates
  2. the algorithm returns the correct value for every input

we propose a statement that describes what the algorithm takes as input and what its supposed to output and then prove it
