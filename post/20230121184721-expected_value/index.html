consider a discrete random variable \(X\) that receives the values \(X_1,X_2,\dots\) with the probabilities \(P_1,P_2,\dots\) respectively, the expected value of \(X\), denoted by \(\mathrm{E}(X)\), is defined as \(\mathrm{E}(X)=\sum_{i} X_iP_i\) such that the series is absolutely convergent

if \(X\) is a constant random variable, i.e. it receives only one value \(C\) with a probability of 1 then \(\mathrm{E}(X)=C\)

if \(a\) is constant, then for each random variable whose expected value is finite \(\mathrm{E}(aX)=a\cdot \mathrm{E}(X)\)

let \(X\) and \(Y\) be two random variables defined over the same probability space, then
\[ \mathrm{E}(X+Y)=\mathrm{E}(X)+\mathrm{E}(Y) \]

let \(X \sim B(n,p)\), we look at \(X\) as the number of successes from a sequence of \(n\) independent trials
