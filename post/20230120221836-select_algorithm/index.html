input: an array \(A\) of \(n\) numbers and a normal number \(k\) such that \(1 \le k \le n\)
output: the element in \(A\) whose rank is \(k\), i.e. the \(k\)’th biggest number in \(A\)
example: for the input \(k=3,A=\{13,15,16,14,12,18,17,11\}\), the output is 13
