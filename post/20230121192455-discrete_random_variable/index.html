discrete random variable

a random variable is a discrete random variable iff it receives a finite or infinite series \(X_1,X_2,\dots\) of the values of the function \(P_x\) that is defined by \(P_x(X_i)=P(X=X_i)\) where \(i=1,2,\dots\) which is called the probability function of the random variable \(X\)

\[ 0 \le P_x(X_i) \le 1 \]

\[ \sum_{i} P_x(X_i) = 1 \]

if \(X\) is a discrete random variable then the list of values that it could receive and their corresponding probabilities are called the distribution law or the probability law of said variable
